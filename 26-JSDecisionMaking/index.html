if statement:
the code in the if block will be evaluated if the condition is true. If statement has the following structure if(condition){code to execute}
if else statement
The if statement gets executed if the condition is true but the else would execute if the condition is false. if(condition){code to execute}else{code to execute}
else if
The esle if statement contain if block that gets executed if the first condition is met, else if block which only gets executed if the first condition is false and the else if condition is true and else condition which gets executed if both conditions are false. if(condition){code to execute}else if(condition){code to execute} else{code to execute}
