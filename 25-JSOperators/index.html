Arithmetic Operators

Arithmetic operators

addition +:
this is used to sum two or more values let sum = 10 + 5;
subtraction -:
this is used to remove one value from another let sum = 10 - 5;
multiplication *:
this is used to multiply two or more values let sum = 10 * 5;
division /:
this is used to divide one value by another let sum = 10 / 5;
modulus %:
this returns the remainder of a division let sum = 10 % 5;
