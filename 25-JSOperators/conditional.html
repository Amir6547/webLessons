Conditional Operators

greater than >:
This checks if the value on the left is greater than the value on the right. It returns either true or false. let isGreaterThan = 10 > 5;
less than >:
This checks if the value on the left is less thanthe value on the right. It returns either true or false. let isLessThan = 10 < 5;
equal to == :
This checks if the value on the left is equal to the value on the right. It returns either true or false. let isEqualTo = 10 == 5;
not equal to != :
This checks if the value on the left is notequal to the value on the right. It returns either true or false. let isNotequalTo = 10 != 5;
greater than or equal to >= :
This checks if the value on the left is greater than or equal to to the value on the right. It returns either true or false. let isGreaterThanOrEqualTo = 10 >= 5;
less than or equal to <= :
This checks if the value on the left is less than or equal to to the value on the right. It returns either true or false. let isLessThanOrEqualTo = 10 <= 5;
strict equal to ===
This does not only check for equality between both values but also checks if they are the same data type. let isStrictlyEqualTo = 10 === '10'; This code will product false as the value on the left is a number while the value on the right is a string. Although they may look the same on the surface but a string and number are not the same data type
