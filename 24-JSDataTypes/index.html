JS Data Types
string:
this can be words or a single character wrapped in single or double quotation. For example, let username = "Amir"
number:
this can be both a whole number (int) or a fractional value (float). For example, let age = 12; const VAT = 7.5
boolean:
these are your true or false values, mostly a result of logical expressions. For example let isActive = true;
undefined:
this is when you create a variable and you do not assign a value to it. For example, let a; console.log(a)
null:
this can represent the absence of a value. For example if you are looking for a value that does not exist in the database. a null value will be returned
symbol:
this is a primitive type just like number or strings but it can only be created using the symbol contructor. Symbols are uniqiue, as no two symbols are the same. For example, const mySymbol = Symbol()
